Easy Ethernet

Visual.net 언어를 사용하여 개발하였으며, ip주소 혹은 도메인 주소를 입력하면 핑을 보내고, 핑이 돌아오는 원리를 이용해 응답속도를 측정하는 프로그램입니다.

측정한 응답속도의 결과값을 엑셀이나 텍스트 형식으로 저장 할 수 있으며, 필터를 이용해 연결이 성공한 결과만 출력하거나, 실패한 결과, 혹은 전체를 출력할 수 있습니다.



* 기록되는 값

1. 측정시간 및 ip주소

2. 결과값 기록
성공시 - OK / 연결성공
실패시 - FAIL / 연결실패
오류시 - UNKNOWN / 오류
작동시 - DEFAULT / 처리중


* 엑셀(csv)과 텍스트(txt)파일로 저장이 가능합니다.
* 오류 : 잘못된 ip주소, 없는 도메인, 인터넷 오류, 부정 요청시
* 최대 1천만번까지 DB 기록이 가능합니다.
* 저사양 모드 체크시 저속 동작하지만, 리소스 사용이 적어집니다.
* 프로그램 구동시 연산오류 경고가 출력되는 경우 컴퓨터의 인터넷 연결 상태를 확인해주세요.
